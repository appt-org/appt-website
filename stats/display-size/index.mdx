In addition to the [zoom function](/en/stats/zoom) and the [enlargement of letters](/en/articles/assistive-technologies/text-enlargement), it is also possible to adjust the display size of other screen items, such as icons. People with less good eyesight can thus make the interface more accessible. This feature is only available on Android devices.

The display size can be increased and decreased with a slider. This feature modifies all elements of the interface: navigation bar, menus, icons, messages, etc. In contrast to zooming in, adjusting the display size works responsively: everything stays neatly within the frame of the screen.

On Android, about 15% percent of users increase the size of the screen. By the way, about 8% reduces the display size.
