# Keyboard access

It is possible to operate your device with an external keyboard. For people with a motor disability, it can be difficult to operate a touchscreen. Pressing keys is easier. For blind people, the keyboard is especially useful to navigate and type faster. There are also many people who connect a keyboard to their tablet. It is also useful for them if they can operate an app via this keyboard.
