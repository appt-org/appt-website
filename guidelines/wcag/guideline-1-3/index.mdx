  icon="/img/light/1.3.svg"
  prefix="Guideline 1.3"
/>

Content on the screen must be able to adapt to users' needs. You can achieve this by exposing all information and properties to the underlying software. The information can then be presented in the desired way. For example, a screen reader can read that bold text is a header.
