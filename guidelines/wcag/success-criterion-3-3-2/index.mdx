Ensure clear instructions are provided when asking users to input data. Add a label to input fields, for example 'First name'. Mark whether fields are required or optional. Indicate if data must be entered in a specific format, such as dates and phone numbers. Clear instructions help all users, especially users with cognitive disabilities.

## Impact

- Clear instructions make entering data more user-friendly for everyone.
- Clear instructions are indispensable for users with cognitive, language and learning disabilities.

## Check

*“Is it clear which data is expected, and which fields are required, inside a forms?”*

This can be tested without assistive technologies.
