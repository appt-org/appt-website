# Scherm automatisch draaien

Op Android is het mogelijk om het automatisch meedraaien van het scherm bij het draaien van de telefoon uit te schakelen. In dat geval behoudt het scherm zijn laatste oriëntatie. Deze functie is belangrijk voor mensen die hun scherm ergens op hebben gemonteerd, zoals een rolstoel. Of fietsers die hun toestel op het stuur vastmaken. 

De meeste apps zijn puur ontwikkeld voor de verticale portretstand. Wanneer het scherm wordt gefixeerd in de horizontale stand, zullen die apps minder goed bruikbaar zijn.

- Op iOS kun je het scherm vergrendelen in de staande stand. Zorg ervoor dat de schermvergrendeling tijdens het testen is uitgeschakeld.
- Op Android kun je het scherm vergrendelen in de meest recente stand. Zorg ervoor dat deze vergrendeling tijdens het testen is uitgeschakeld.

Op dit moment meten we nog geen schermoriëntatie op iOS.
