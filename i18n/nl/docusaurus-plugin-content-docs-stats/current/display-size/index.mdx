# Weergavegrootte

Naast de [zoomfunctie](/nl/stats/zoom) en het [vergroten van letters](/nl/artikelen/hulpmiddelen/tekstvergroting) is het ook mogelijk om de weergavegrootte van andere schermitems aan te passen, zoals pictogrammen. Mensen met minder goed zicht kunnen zodoende de interface toegankelijker maken. Deze functie is alleen beschikbaar op Android-toestellen.

De weergavegrootte kan op Android met een schuifregelaar vergroot en verkleind worden. Deze functie past alle elementen van de interface aan: navigatiebalk, menu's, pictogrammen, berichten, et cetera. In tegenstelling tot inzoomen werkt weergavegrootte aanpassen responsive: alles blijft keurig binnen het kader van het scherm.

Op Android zet zo'n 15% procent van de gebruikers de weergave van het scherm groter. Overigens verkleint ongeveer 8% de weergavegrootte.
