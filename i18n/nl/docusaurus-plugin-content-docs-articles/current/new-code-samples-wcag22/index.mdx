# Nieuwe codevoorbeelden voor WCAG 2.2

We zijn enthousiast om aan te kondigen dat Stichting Appt nieuwe codevoorbeelden heeft uitgebracht voor WCAG 2.2. Codevoorbeelden voor Swift en SwiftUI zijn nu beschikbaar voor de succescriteria van niveau A en AA.

- [2.4.11 Focus niet bedekt (minimum)](https://appt.nl/2.4.11)
- [2.5.7 Sleepbewegingen](https://appt.nl/2.5.7)
- [2.5.8 Grootte van het aanwijsgebied (minimum)](https://appt.nl/2.5.8)
- [3.2.6 Consistente hulp](https://appt.nl/3.2.6)
- [3.3.7 Overbodige invoer](https://appt.nl/3.3.7)
- [3.3.8 Toegankelijke authenticatie (minimum)](https://appt.nl/3.3.8)
