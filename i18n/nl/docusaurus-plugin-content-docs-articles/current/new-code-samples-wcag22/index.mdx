# Nieuwe codevoorbeelden voor WCAG 2.2

We zijn verheugd aan te kondigen dat de Appt Foundation nieuwe codevoorbeelden heeft uitgebracht voor WCAG 2.2. Codevoorbeelden voor Swift en SwiftUI zijn nu beschikbaar voor de succescriteria van niveau A en AA.

- [2.4.11 Focus Not Obscured](https://appt.nl/2.4.11)
- [2.5.7 Dragging Movements](https://appt.nl/2.5.7)
- [2.5.8 Target Size (Minimum)](https://appt.nl/2.5.8)
- [3.2.6 Consistent Help](https://appt.nl/3.2.6)
- [3.3.7 Redundant Entry](https://appt.nl/3.3.7)
- [3.3.8 Accessible Authenticatio (Minimum)](https://appt.nl/3.3.8)

## Help ons met uitbreiden

We hebben hulp nodig om deze codevoorbeelden uit te breiden naar Android, Jetpack Compose, Flutter, React Native, .NET MAUI en Xamarin.

Indien je kennis hebt van een van deze frameworks, vragen we je om je kennis met ons te delen.

In ons artikel [Deel codevoorbeelden voor toegankelijke apps](/nl/artikelen/deel-codevoorbeelden-toegankelijke-apps) kun je meer lezen over het proces om bij te dragen.

De 6 nieuwe codevoorbeelden zijn hieronder ingevoegd. Kies je framework en druk op de knop "Bijdragen" om de locatie van het bestand in onze open-source repository te vinden.
