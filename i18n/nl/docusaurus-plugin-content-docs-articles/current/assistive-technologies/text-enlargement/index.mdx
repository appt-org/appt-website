# Hoe werkt tekstvergroting in een app?

Op de telefoon kun je bij de systeeminstellingen tekstvergroting aanzetten. Het aanzetten werkt op elk toestel anders. Door te zoeken op “tekst” kun je de optie meestal vinden. Door de tekstgrootte op maximaal te zetten kun je zien of jouw app tekstvergroting ondersteund en of dit problemen veroorzaakt.

- **Android**: Instellingen → Scherm → Lettergrootte
- **iOS**: Instellingen → Toegankelijkheid → Weergave en tekstgrootte → Grotere tekst
