Darice werkt als front-end developer een heeft een passie voor inclusief design. Ze geeft ook lezingen geeft over inclusief design en toegankelijkheid. In haar dagelijks leven gebruikt ze zowel een iPhone als een Android-telefoon. Die laatste heeft Darice vanwege Googles [Live Transcribe](/nl/stats/ondertiteling)-app, een functionaliteit die nog niet op de iPhone ondersteund werd ten tijde van dit interview. Tegenwoordig gebruikt ze ook een [Apple Watch](https://www.darice.org/2022/11/25/apple-watch-and-accessibility-for-the-deaf).

## Welke toegankelijkheidsfuncties heb je op je telefoon ingesteld?

Trillen bij binnenkomende berichten heb ik echt nodig, anders mis ik ze gegarandeerd. Maar het trillen op nieuwere iPhones is minder sterk geworden. Er is geen manier om het harder te zetten, dus ik heb nu ook een flits erbij nodig, zeker bij een alarm. Dan kan ik het zien in plaats van voelen.

Dit is echt noodzakelijk voor mij, anders vergeet ik afspraken. Bij SMS'jes en WhatsApp licht het scherm op, zodat ik weet dat ik een berichtje krijg. Voor een alarm zet ik de flits aan. Die is zo sterk, dat wanneer ik mijn telefoon met het scherm op tafel heb liggen, de flits zelfs door het mobieltje heen gaat. 

Siri is ook handig, want Siri reageert ook met tekst. Siri gebruikte ik eerst, maar ik vind het gek om tegen een telefoon te praten en door mijn accent verstaat Siri mij niet altijd.

iPhones hebben ook een instelling met een luisterend oor-icoon, waarmee je kan instellen dat je gewaarschuwd wordt bij bepaalde geluiden. Dit had ik eerst aan staan, maar het werkt zo goed dat ik heel veel meldingen kreeg. In de stad is er bijvoorbeeld altijd wel een ambulance, wat deze instelling dan triggert. Omdat het storend werd, heb ik het uitgezet.

Naast mijn mobiel maak ik gebruik van een schrijftolk bij vergaderingen en interviews. Vergaderingen zijn belangrijk, maar ik kan niet steeds een schrijftolk vragen. Dus sommige vergaderingen of gesprekken gaan via een WhatsApp-groep. Het continu thuiswerken tijdens corona was super handig voor iemand die doof is, want alles ging via e-mail en WhatsApp-groepen.
