## Welke toegankelijkheidsfuncties heb je op je telefoon ingesteld?

De instellingen voor toegankelijkheid zijn best verstopt, vind ik. Ik heb de [lettergrootte](/nl/stats/lettergrootte) aangepast (op ⅓ van het balkje), dan kan ik zonder bril lezen. Als ik even een bericht wil bekijken, vergroot ik het ook op een website bijvoorbeeld. In apps is het vaak niet mogelijk om te [zoomen](/nl/stats/zoom). 

De grootte van teksten aanpassen is voor mij noodzakelijk. Ik ben er echt op zoek naar gegaan, toen mijn ogen slechter werden. Een leesbril is niet handig: ik zet die steeds op en af. Op mijn mobiel zie ik mét leesbril beter, maar alles verder weg is dan wazig, dus houd ik ‘m niet op.

Het [contrast](/nl/stats/verhoogd-contrast) heb ik overigens ook hoger staan om de leesbaarheid te verbeteren. Op mijn mobiel heb ik altijd light mode aanstaan. Ik vind teksten beter leesbaar op een witte achtergrond. [Dark mode](/nl/stats/donkere-modus) vind ik niet fijn in gebruik, de tekst is dan voor mij niet goed leesbaar.

Overigens is het voor mijn ogen fijn als een scherm groter is. Smartphones worden de laatste jaren steeds groter, maar grotere modellen kan ik amper vasthouden. Ik heb niet zulke grote handen en wil mijn telefoon wel met één hand kunnen bedienen. Voor mij gaat toegankelijkheid ook om het vasthouden van een telefoon en het bedienen daarvan.
