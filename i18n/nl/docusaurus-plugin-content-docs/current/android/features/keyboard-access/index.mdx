# Toetsenbordbediening voor Android

Het is mogelijk om je Android toestel te bedienen met een extern toetsenbord. Door toetsen in te drukken op het toetsenbord kun je navigeren en acties uitvoeren. Toetsenbordbediening is beschikbaar op Android telefoons en tablets.

<Image src="/img/light/android-tablet-met-toetsenbord.png" alt="Android tablet with keyboard - Keyboard access"/>

*Android tablet met aangesloten toetsenbord*

## Toetsenbord aansluiten

Het aansluiten van een toetsenbord is de eerste stap om te beginnen met toetsenbordbediening.

Voor Android raden wij aan om een Windows toetsenbord aan te sluiten. Je kunt echter ook een Mac toetsenbord aansluiten. In onderstaande tabel kun je zien hoe de toetsen zich tot elkaar verhouden.

| Windows | Mac |
| --- | --- |
| Windows | Command |
| Ctrl | Control |
| Alt | Option |

*Verschil tussen toetsen op Windows en Mac.*

Je kunt het toetsenbord op twee manieren aansluiten: via Bluetooth of via de USB-poort.

### Manier 1: via Bluetooth

1. Open de** Instellingen-app**
2. Scroll naar onderen, tik op** Verbonden apparaten**
3. Tik op** Apparaat koppelen** (Bluetooth wordt automatisch ingeschakeld)
4. Zet je Bluetooth-toetsenbord aan
5. Wacht tot het toetsenbord onder** Beschikbare apparaten** verschijnt
6. Selecteer het toetsenbord om te beginnen met verbinden
7. Er verschijnt een pop-up, selecteer de optie** Koppelen**

Als het goed is staat je toetsenbord nu onder het kopje **Momenteel verbonden.**

### Manier 2: via USB-poort

Het toetsenbord sluit je aan door de kabel in de poort te stoppen. De aansluiting van het toetsenbord moet hetzelfde zijn als de poort van je toestel. Meestal is dit niet het geval, je moet dan een verloopstukje aanschaffen.

Controleer eerst welke USB-poort je toestel heeft. Nieuwe toestellen hebben een USB-C poort, oudere toestellen hebben een Micro-USB poort.

<Image src="/img/light/usb-c-poort.png" alt="USB-C port - Keyboard access"/>

*USB-C poort*

<Image src="/img/light/micro-usb-poort.png" alt="Micro-USB port - Keyboard access"/>

*Micro-USB poort*

Controleer vervolgens de aansluiting van het toetsenbord. Het meest voorkomend is een normale USB aansluiting, je hebt dan een verloopstukje nodig. Er zijn ook toetsenborden speciaal voor Android met USB-C of Micro-USB aansluiting.

## Bediening via toetsenbord

Nu je het toetsenbord hebt aangesloten vraag je je vast af hoe je nu je toestel kunt bedienen. Het belangrijkste om te weten zijn:

- **Tab**: navigeren naar volgende element
- **Pijltjestoetsen**: navigeren door lijsten
- **Spatie**: geselecteerde element activeren

Een app zou volledig te bedienen moeten zijn met deze drie toetsen. Helaas houden de meeste apps geen rekening met toetsenbordbediening. Het zal vaak voorkomen dat je apps niet volledig kunt bedienen met het toetsenbord.

In onderstaande tabellen vind je toetsencombinaties die beschikbaar zijn.

### Overzicht

Je kunt een overzicht van beschikbare sneltoetsen tonen. Op Windows toetsenborden druk je hiervoor de **Windows + /** in. Op Mac toetsenbord gebruik je de toetsen **Command + /**. Bij het overzicht staat het zoekglaasje respectievelijk voor de Windows en Command toets.
