# Appt app

Bijna de helft van alle mensen met een mobiele telefoon gebruikt toegankelijkheidsfuncties. Véél meer dan je zou verwachten, toch?

Helaas houden de meeste apps geen rekening met deze toegankelijkheidsfuncties. Als gevolg hiervan kunnen miljoenen mensen geen apps gebruiken.

Download de Appt app en leer je hoe je jouw app toegankelijk kunt maken!
