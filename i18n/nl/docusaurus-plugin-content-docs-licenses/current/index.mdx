# Licenties

Informatie op Appt.org is gratis beschikbaar onder verschillende licenties. Wanneer je gebruik maakt van onze informatie vragen we je om een link op te nemen naar de bron. Hierdoor vinden meer mensen hun weg naar ons om te leren hoe ze apps toegankelijker kunnen maken.

De onderstaande informatie is mogelijk verouderd. Raadpleeg het [LICENSE bestand](https://github.com/appt-org/appt-website/blob/HEAD/LICENSE) voor de huidige licenties.

## Inhoud

Tenzij anders aangegeven, is inhoud beschikbaar onder de voorwaarden van de [CC BY-SA licentie](https://creativecommons.org/licenses/by-sa/4.0/).

## Code

Tenij anders aangegeven, is code beschikbaar onder de voorwaarden van de [MIT licentie](https://opensource.org/licenses/MIT).

## Publicatie vereisten

Bij het publiceren van onze content of code moet je:

- Adequate bronvermelding verstrekken zoals vereist door CC BY-SA 4.0 en MIT.
- Een zichtbare link naar `https://appt.org` of `https://github.com/appt-org/appt-website` opnemen op elke pagina waar onze content gebruikt wordt.

## Alternatieve licenties

Als je niet kunt voldoen aan de vereisten, neem dan contact met ons op via [info@appt.org](mailto:info@appt.org) om alternatieve licentiemogelijkheden te bespreken, waaronder white-label.
