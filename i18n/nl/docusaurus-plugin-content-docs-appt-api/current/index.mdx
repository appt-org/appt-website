# Appt API

Wil je onze content integreren in jouw product? Dat kan door gebruik te maken van onze API!

Onze website wordt mogelijk gemaakt door [Contentful](https://www.contentful.com), een composable content platform. Je kunt de [Content Delivery API](https://www.contentful.com/developers/docs/references/content-delivery-api/) gebruiken om alle inhoud van deze website op te halen.

Let op: bij het gebruik van onze inhoud moet je rekening houden met onze [licenties](/nl/licenties).

Stuur een e-mail naar [info@appt.org](mailto:info@appt.org) om een API-sleutel aan te vragen.
