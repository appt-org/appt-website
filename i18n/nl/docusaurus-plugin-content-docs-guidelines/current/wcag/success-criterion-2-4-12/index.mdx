Zorg dat elementen volledig zichtbaar blijven bij toetsenbordfocus. Wanneer het geselecteerde element niet zichtbaar is, weten gebruikers mogelijk niet hoe ze verder moeten gaan. Het gefocuste element mag niet bedekt worden door andere inhoud.
