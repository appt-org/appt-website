# Toegankelijke authenticatie (minimum)

Zorg dat het makkelijk is om in te loggen. Gebruikers moeten een wachtwoordmanager kunnen gebruiken of hun wachtwoord kunnen kopiëren en plakken. Voor gebruikers met  cognitieve beperkingen kan het moeilijk zijn om wachtwoorden te onthouden.
