Zorg dat elementen ten minste gedeeltelijk zichtbaar blijven bij toetsenbordfocus. Wanneer het geselecteerde element niet zichtbaar is, weten gebruikers mogelijk niet hoe ze verder moeten gaan. Het gefocuste element mag niet volledig bedekt worden door andere inhoud.
